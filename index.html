Synchronisation (Google Drive / iCloud)

Pour synchroniser entre Windows et iPhone, place le fichier facturation.json dans ton Google Drive ou iCloud Drive. L'app lit et écrit ce fichier à chaque fois.

Installer sur iPhone (Safari)
  • Ouvre cette page dans Safari sur ton iPhone
  • Appuie sur le bouton Partage (rectangle avec flèche vers le haut)
  • Appuie sur « Sur l'écran d'accueil »
  • L'icône apparaît comme une vraie app
Installer sur Windows (Chrome / Edge)
  • Ouvre cette page dans Chrome ou Edge
  • Clique sur l'icône d'installation dans la barre d'adresse (ou menu → Installer)
  • L'app apparaît dans le menu Démarrer et le bureau
  • Elle fonctionne même hors-ligne
Données

Toutes tes données sont stockées localement sur cet appareil. Exporte régulièrement vers ton Drive pour les synchroniser.
